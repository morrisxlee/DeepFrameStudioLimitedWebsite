# 公司 Logo 資料夾 / Company Logo Folder

將公司 Logo 放入此資料夾。
Place your company logo files in this folder.

建議放入以下檔案 / Recommended files:
- `logo.png` — 主要 Logo / Primary logo
- `logo-dark.png` — 深色版本 / Dark version
- `logo-light.png` — 淺色版本 / Light version
- `favicon.ico` — 網站圖示 / Website favicon

放入後可以在組件中引用，例如：
After adding, reference in components like:

```html
<img src="/assets/logo/logo.png" alt="Deep Frame Studio" />
```
