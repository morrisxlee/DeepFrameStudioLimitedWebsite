# 媒體素材資料夾 / Media Assets Folder

將相關報道的圖片放入此資料夾。
Place media coverage images in this folder.

支援格式 / Supported formats: `.jpg`, `.png`, `.webp`, `.svg`

放入後可以在 `src/app/[locale]/media/page.tsx` 中引用，例如：
After adding images, reference them in `src/app/[locale]/media/page.tsx` like:

```html
<img src="/assets/media/your-image.jpg" alt="description" />
```
